Objeto de Documento HTML - DOM

O DOM é o responsável por todos os objetos de documento Html na sua página.
Existem métodos para todo tipo de manipulação de valores com o DOM, alterações, seleções, exclusões e etc...

Localizando Elementos
  • document.getElementById("nome-id") - Localiza um elemento pelo ID
  • document.getElementByTagName("nome-tag") - Localiza um elemento pela TAG
  • document.getElementsbyClassName("nome-classe")
Alterando Elementos
  • nome-elemento.innerHTML = 'novo-conteudo' - Altera o elemento interno
  • nome-elemento.attribute = 'novo-valor' - Altera o valor do atributo de um elemento
  • nome-elemento.style.property = 'novo-estilo'Altera o estilo de um elemento
  • nome-elemento.setAttribute(atributo, valor) - Altera o valor do atributo de um elemento
Adicionando Elementos
  • document.createElement(elemento) - Cria um elemento
  • document.appendChild(elemento) - Adiciona um elemento
  • document.write('texto') - Exibe o texto que foi escrito
Deletando Elementos
  • document.removeChild(elemento) - Remove um elemento
  • document.replaceChild(novo, antigo) - Substitui o elemento
Selecionando Elementos CSS
  • document.qerySelectorAll('nome-tag.nome-elemento') - Vai selecionar todos os elementos indicados com um atributo CSS (id, classe, tipo, valor de atributo e etc...)
Eventos com JavaScript
  • onclick='nome-evento' - Vai permitir manipular determinadas ações criando ações com cliques
Manipulador de Eventos
  • document.getElementById('nome-id').onclick = função(){nome-codigo} - Cria um manipulador de eventos no qual chama uma função ao ser clicado
