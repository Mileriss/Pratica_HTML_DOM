Métodos DOM

Quando utilizamos o DOM todos os elementos HTML são considerados Objetos e deta forma é possível manipularmos esses objetos utilizando JavaScript. As propriedades DOM são valores de elementos HTML que você pode definir, alterar ou manipular da forma que desejar.
Neste exemplo será utilizado um método (GetElementById) que vai interagir com um valor (innerHTML)
